## 💡 About This Project & What I Learned Today

1. **Day 4 Focus:** Developed an interactive Dev Card Generator contained in a single `index.html` file, moving beyond static layouts into functional DOM state management.
2. **UI & Styling:** Applied a custom dark-mode theme where form inputs dynamically render text in a sleek grey font (`#94a3b8`) for better visual hierarchy.
3. **Form Logic:** Handled multiple HTML5 input types—including text fields, dropdown selectors, radio groups, and boolean checkboxes—to capture structured user data.
4. **DOM Manipulation:** Replaced default browser behavior using `event.preventDefault()` to dynamically extract form values and inject them into a newly generated UI component.
5. **State Toggling:** Implemented interactive view-switching in JavaScript that hides the input form upon completion and displays the final card, complete with a reset function.
